# Python Portfolio

This repository contains beginner-to-intermediate Python projects
built while learning software development and AI fundamentals.

## Projects

### Calculator
A command-line calculator supporting basic arithmetic.
- Concepts: functions, conditionals, input handling

### Contact Book
A menu-driven contact manager.
- Concepts: dictionaries, loops, functions

### To-Do List
A persistent to-do list using file storage.
- Concepts: file I/O, error handling

### Expense Tracker
Tracks and summarizes expenses.
- Concepts: data structures, basic reporting
